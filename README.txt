To run cd into the 'dist' folder and run
java -jar huge-drawing-tool-assembly-1.0.jar

You can also pass in all the arguments. Running the program like this won't let the user interact with it inside, and will terminate.
java -jar huge-drawing-tool-assembly-1.0.jar  "C 20 4" "L 1 2 6 2" "L 6 3 6 4" "R 16 1 20 3" "B 10 3 o"

The program was tested with java 8. If you have any questions, email me at roger.planetgreen@gmail.com
